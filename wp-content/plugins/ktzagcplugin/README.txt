== Description ==

This is free agc plugin from kentooz, that usefull for boost your traffic and get more index in search engine.

== Installation ==

1. Upload `ktzagcplugin` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Done, setting and ready to use.
